# java-chess 게임

체스 게임 구현
체스판은  
왼쪽에서 오른쪽으로 a~h  
아래에서 위로 1~8로 구현

기능 1. 말들을 앞으로 1칸씩 움직일 수 있다.  
기능 2. 움직인 말은 상대 말을 잡아먹을 수 있다.  
기능 3. move source위치 target위치 와 같은 명력어로 움직인다
